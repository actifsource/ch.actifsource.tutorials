Service Design

The Service consists of Service Calls which might have zero to many arguments.

class Service extends NamedResource
Class Description
This Service can be used via the Calls provided.
 
name : StringLiteral[1..1]
 Property Description
The name of this Resource.
 
call : Call[1..N]
 Property Description
Every Service consists of least one Call.
